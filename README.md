# 基于Django的Python课程学习资源推送系统 python1239

## 项目概述

本项目是一款基于Django框架的Python课程学习资源推送系统，旨在为用户提供优质的学习资源，并通过系统化的推送，提高学习效率。

## 技术栈

- **后端框架：** Django
- **编程语言：** Python 3.7/3.8
- **数据库：** MySQL 5.7 或更高版本
- **数据库工具：** Navicat 11（推荐）
- **开发工具：** PyCharm / VS Code
- **前端框架：** Vue.js

## 功能模块

- 资源推送
- 课程学习
- 用户管理
- 资源管理

## 系统角色

- 管理员
- 用户

## 运行环境

- Python 3.7/3.8
- Django
- MySQL 5.7 或更高版本
- Vue.js

## 部署步骤

1. 环境准备：安装 Python、Django、MySQL 和 Vue.js
2. 数据库配置：使用 Navicat 11 创建数据库
3. 系统部署：配置系统环境，运行项目

## 目录结构

```
├── app             # 应用代码目录
│   ├── admin       # 管理后台
│   ├── models      # 模型定义
│   ├── views       # 视图处理
│   └── ...
├── static          # 静态文件目录
├── templates       # 模板文件目录
├── ...
```

- 基于Django框架，稳定可靠
